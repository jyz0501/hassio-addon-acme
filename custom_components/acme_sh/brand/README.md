# Brand Images

This directory contains brand images for the ACME.sh SSL Certificate Manager integration.

## Supported Files

Place the following image files in this directory:

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `icon.png` | Integration icon (light theme) | 512x512 px |
| `icon@2x.png` | Integration icon (retina) | 1024x1024 px |
| `dark_icon.png` | Integration icon (dark theme) | 512x512 px |
| `dark_icon@2x.png` | Integration icon (dark, retina) | 1024x1024 px |
| `logo.png` | Integration logo (light theme) | 512x512 px |
| `logo@2x.png` | Integration logo (retina) | 1024x1024 px |
| `dark_logo.png` | Integration logo (dark theme) | 512x512 px |
| `dark_logo@2x.png` | Integration logo (dark, retina) | 1024x1024 px |

## Requirements

- Images must be in PNG format
- Use transparent backgrounds for icons
- Recommended to provide both light and dark theme variants
- Images should be square

## Notes

- Local brand images automatically take priority over images from the brands CDN
- This feature requires Home Assistant 2026.3 or later
- For older versions, images will not be displayed
